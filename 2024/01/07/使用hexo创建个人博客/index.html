使用hexo创建个人博客

Created2024-01-07|Updated2024-01-08
|Post Views:

初始化

安装Nodejs

https://nodejs.org 官网
作者选择的是左边长期支持版
下载并安装(next即可)

打开终端分别输入 node -v 和 npm -v 查看node版本和npm版本，显示版本号即成功安装

安装cnpm

npm是从rejistry仓库中获取的，但是由于rejistry仓库服务器位于国外就会导致包的获取速度较慢，所以先利用npm安装淘宝的镜像服务器cnpm，使用cnpm后就会直接从淘宝仓库获取包。
npm install -g cnpm –registry=http://registry.npm.taobao.org 安装cnpm
cnpm -v 查看cnpm版本

安装hexo

cnpm install -g hexo-cli 安装hexo框架
hexo -v 查看hexo版本

mkdir blog 创建blog目录
cd blog 进入blog目录
hexo init 生成博客 初始化博客
hexo s 启动本地博客服务
http://localhost:4000/ 本地访问地址，可以在浏览器上查看你的博客了

创建新文章

hexo n “我的第一篇文章” 创建新的文章
新文章默认保存在C:/用户/当前用户/blog/source/_posts
可以使用记事本、vscode、typora等对其进行编辑

部署到github

在Github上创建一个新的仓库 YourGithubName.github.io

cnpm install –save hexo-deployer-git 使用终端在blog目录下安装git部署插件

配置_config.yml

对blog目录下的_config.yml文件做出如下修改

# Deployment
## Docs: https://hexo.io/docs/deployment.html
deploy:
      type: git
     repo: https://github.com/YourGithubName/YourGithubName.github.io.git
      branch: main

hexo deploy 部署到Github仓库里
https://YourGithubName.github.io/ 访问这个地址就可以查看博客

美化

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia 下载yilia主题到本地

修改blog目录下的 _config.yml 文件： theme: yilia

hexo clean 清理
hexo generate 生成
hexo s 预览效果
hexo deploy 重新部署到远程Github仓库，部署有一定的延迟
https://YourGithubName.github.io/ 查看博客
